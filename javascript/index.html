什么是JavaScript

JavaScript ( JS ) 是一种轻量级解释型的编程语言。虽然它是作为开发web页面的脚本语言而出名的， 但是在很多非浏览器环境中也使用JavaScript，例如 node.js。

JavaScript 和 ECMAScript

ECMAScript ( ES ) 是 JavaScript 的标准。JavaScript是ECMAScript的标准的实现和扩展。 当前的JavaScript版本是ECMAScript 5，ECMAScript 6当前是开发版。

JavaScript实现

一个完整的JavaScript实现应该由下列三个部分组成：

核心 ( ECMAScript ) ： 提供核心语言功能

文档对象模型 ( DOM ) ： 提供访问和操作网页内容的方法和接口

浏览器对象模型 ( BOM ) ： 提供和浏览器交互的方法和接口
